Portafolio Argentina-Programa

El proyecto consistía en armar una aplicación web full stack que mostrará mis datos personales, los estudios cursados, la experiencia laboral, el conocimiento de las tecnologías y demás cosas que se deseen agregar. En pocas palabras un portafolio web.

El portafolio es de arquitectura distribuida y contiene un diseño de interfaz de usuario que muestra la información, una base de datos que almacena los datos antes mencionados y contiene la APIs necesarias para proveer a través de internet la información.

A través de los distintos modulos planteados en el curso, se iba avanzando en el desarrollo del portafolio. Primero fue la parte de frontend, enmaquetado utilizado los lenguajes de HTML, CSS y JavaScrip. Además, se conto con la ayuda de frameworks como bootstrap o angular para hacer la aplicación dinámica.
